Projeto simples para fazer download de músicas em .mp3 usando bot do Telegram.

O bot conta com:

- Download de arquivos do YouTube usando a biblioteca yt-dlp;
- Conversão de arquivos m4a para mp3 usando a biblioteca ffmpeg;
- Salvamento das músicas na pasta downloads e possibilidade de baixá-las novamente com o comando /lista;
- Limpeza da pasta de downloads com o comando /limpar;
- Configuração fácil para quem for usar;
- Recuperação da fila de downloads caso o terminal seja fechado;
- Armazenamento de músicas separado por ID do usuário do Telegram.


  Instruções de configuração:
- Adicione a chave de acesso do bot do Telegram no arquivo configBot.js.
- Adicione o ID do chat do administrador no arquivo bot.js (variável ADMIN_ID)

Demo: https://t.me/YoutbMother_bot
-----------------------------------------------

Inglish:

A simple project to download music in .mp3 format using a Telegram bot.

The bot includes:

- Downloading files from YouTube using the yt-dlp library;
- Converting .m4a files to .mp3 using the FFmpeg library;
- Saving songs in the downloads folder and allowing them to be downloaded again using the /lista command;
- Cleaning the downloads folder using the /limpar command;
- Easy configuration for anyone who wants to use it;
- Recovery of the download queue if the terminal is closed;
- Storage of songs separated by each Telegram user's ID.

Setup instructions:
- Add your Telegram bot access key in the configBot.js file.
- Add the administrator chat ID in the bot.js file (variable ADMIN_ID).
